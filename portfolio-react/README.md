# Personal Portfolio 🚀

This is my **personal developer portfolio** built using **React +
Vite**.\
It showcases my projects, skills, and contact information.

## 🔗 Live Website

Once deployed with Vercel, the portfolio will be available online.

## 🛠 Tech Stack

-   **React**
-   **Vite**
-   **JavaScript**
-   **HTML5**
-   **CSS3**
-   **Git & GitHub**
-   **Vercel (Deployment)**

## 🚀 Getting Started

Clone the repository:

    git clone https://github.com/AashishGupta2007/portfolio.git

Go into the React project folder:

    cd portfolio/portfolio-react

Install dependencies:

    npm install

Run the development server:

    npm run dev

Open the local server shown in the terminal (usually):

    http://localhost:5173

## 📦 Build for Production

To create an optimized production build:

    npm run build

The production files will be generated in the **dist/** folder.

## 🌐 Deployment

This portfolio is deployed using **Vercel**.

Steps:

1.  Push project to GitHub
2.  Import repository into Vercel
3.  Set **Root Directory** to:
